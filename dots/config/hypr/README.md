# Hyprland

Ah, hyprland it's such a noice piece of software and it really made the switch to Wayland easy. That all being said the fact there are so many other wonderful projects out there, and I'm more used to a dynamic tiling window manager, once I discovered I could switch without too much issue I well... did.

![](/scrot/scrot.png)

### The Switch

I have migrated for the time being. With all the tools used to turn a basic window manager into a more full setup all being modular and separate it's actually pretty easy to port the bar, hotkeys, menus, and the like over making this the least painful transition I've ever had.

## MangoWC

I'm liking where I've landed and it feels like a full circle. Hyprland is a lot like i3, as far as I understand it, in that window position is picked as things are created by what is active. I like the Dynamic Master and Stack but the layouts can be switched way of thinking about things plus Tags over Workspaces so this was reason enough to switch. With Wayland's compositor being part of the program my Hyprland rice was immiedately available with all the blur and whatnot. Hyprland paved the way for those things to be easy, so I'm ever grateful for such a cool project.
